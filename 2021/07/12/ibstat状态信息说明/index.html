ibstat状态信息说明

发表于2021-07-12|更新于2021-07-12|运维
|阅读量:

简介

通过 ibstat 命令查看IB HCA的状态。

输出格式例:

Physical State

Physical State （物理状态）指示线缆的状态，这个字段有两种值：

Polling

轮询状态，从这张卡到另一张卡没有连接时会显示。

LinkUp

与另一端设备已连接。仅表示在物理层已经连接。

State

State (状态)指示HCA端口是否启动，以及是否已被子网管理发现。

Down

HCA卡和线缆另一端的设备之间没有物理连接。当 Physical State 为 Polling 时，一般都会是这个状态。

Initializing

初始化中。指示HCA卡和线缆另一端的设备已经建立物理连接，但是子网管理尚未发现它，需要确保有一个托管交换机，或者集群中有运行着的 opensm 进程。

Active

端口处于正常运行状态。

Rate

指示端口运行的速度。
